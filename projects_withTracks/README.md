# 實作專案區 (Projects with Tracks)

> 對應各個 tracks 的練習專案
> ⚠️ **規則: 這裡的專案不能依靠 AI 生成程式碼，要自己手寫**

---

## 📁 資料夾對應

| 資料夾 | 對應 track | 專案範例 |
|--------|-----------|----------|
| `php_oop/` | PHP OOP | 自己寫一個 User class |
| `mysql_advanced/` | MySQL 進階 | 優化一個慢查詢 |
| `data_structures/` | 資料結構 | 手刻 Stack/Queue |
| `algorithms/` | 演算法 | 不查資料寫 Binary Search |
| `redis/` | Redis | 實作一個快取功能 |
| `docker/` | Docker | 容器化一個專案 |
| `linux/` | Linux | 寫一個 shell script |
| `ai_llm/` | AI & LLM | 做一個 Chatbot |

---

## 為什麼不能用 AI？

你在 test.md 說過：「Claude Code 開發，我用眼睛看」

這個區域是為了訓練你的**手刻能力**：
- 腦中建立程式模型
- 培養 Debug 直覺
- 面試時能手寫程式

---

## 專案模板

每個專案資料夾建議有：
```
專案名稱/
├── README.md      # 專案說明、目標、學到什麼
├── src/           # 程式碼
└── notes.md       # 開發過程筆記
```
