# 學習資源區 (Resources with Tracks)

> 對應各個 tracks 的學習資源整理

---

## 📁 資料夾對應

每個資料夾對應一個 track，裡面放該主題的：
- 🔗 網站連結
- 📚 書籍推薦
- 🎬 影片課程
- 📝 文章收藏

---

## 資源模板

每個 track 的資源檔案建議格式：

```markdown
# [主題名稱] 學習資源

## 免費資源

### 網站
- [資源名稱](連結) - 簡短說明

### 影片
- [影片名稱](連結) - 簡短說明

## 付費資源

### 課程
- [課程名稱](連結) - 價格、評價

### 書籍
- 《書名》作者 - 簡短評價

## 我的評價

學完後可以回來補充哪個資源最有用。
```
