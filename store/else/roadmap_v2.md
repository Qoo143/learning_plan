# 學習路線圖 v2 (精簡可執行版)

> 設計原則: 聚焦、可執行、工作優先、AI 並行
> 每週投入: 平日 5hr + 假日 4hr = **約 9 小時/週**

---

## 🎯 整體策略

```
             工作主線 (60%)              興趣副線 (40%)
                 │                           │
    ┌────────────┴────────────┐              │
    │                         │              │
 Phase 1: 補強           Phase 2: 進階       │
 PHP OOP + MySQL         架構 + DevOps    Python → LLM → Agent
 (3 個月)                (6 個月)          (持續並行)
```

**規則**:
- 平日專注「工作主線」(PHP/MySQL)
- 假日可以玩「興趣副線」(AI)
- 不貪多，一次只學一個主題

---

## 📅 Phase 1: 基礎補強 (2025/12 - 2026/02, 約 12 週)

> 目標: 把工作相關技能從「能用」提升到「懂原理」

### Sprint 1-4: PHP 深化 (第 1-4 週)

| 週次 | 主題 | 具體內容 | 產出 |
|------|------|----------|------|
| W1 | OOP 基礎 | class, 繼承, 介面, 抽象類 | 重構一個工作中的 function 成 class |
| W2 | OOP 進階 | trait, namespace, autoload | 理解公司專案的 autoload 機制 |
| W3 | 設計模式入門 | Singleton, Factory, Strategy | 在工作專案中找出這些模式 |
| W4 | 錯誤處理 | Exception, 自定義例外, 日誌 | 改善工作專案的錯誤處理 |

**學習資源**:
- [ ] PHP The Right Way (免費線上)
- [ ] 工作專案源碼 (最好的教材)

---

### Sprint 5-8: MySQL 強化 (第 5-8 週)

| 週次 | 主題 | 具體內容 | 產出 |
|------|------|----------|------|
| W5 | JOIN 精通 | INNER, LEFT, RIGHT, 多表連接 | 複雜報表查詢練習 |
| W6 | 索引原理 | B-Tree, 複合索引, EXPLAIN | 分析工作專案的慢查詢 |
| W7 | 查詢優化 | 避免 N+1, 子查詢 vs JOIN | 優化一個工作中的慢 API |
| W8 | 交易與鎖 | Transaction, 死鎖, 隔離級別 | 理解工作專案的交易處理 |

**學習資源**:
- [ ] 工作環境的慢查詢日誌
- [ ] MySQL 官方文檔 - Optimization

---

### Sprint 9-12: 理論補課 (第 9-12 週)

| 週次 | 主題 | 具體內容 | 為什麼要學 |
|------|------|----------|------------|
| W9 | HTTP 深入 | 狀態碼, Header, 快取機制 | Debug API 問題 |
| W10 | 資料結構概念 | Array, Hash Table 原理 | 理解 PHP 陣列底層 |
| W11 | 演算法思維 | Big O, 常見複雜度 | 評估程式效能 |
| W12 | 總結複習 | 回顧 + 查漏補缺 | 鞏固知識 |

## 📅 Phase 2: 進階能力 (2026/03 - 2026/08, 約 24 週)

> 根據 Phase 1 的學習狀況調整，這裡先列大方向

| 月份 | 主題 | 關鍵技能 |

## 🤖 AI 副線 (並行進行，假日學)

> 這條線是你的興趣，保持熱情最重要

### Level 1: Python 基礎 (隨時開始)
- [ ] Python 語法熟悉
- [ ] 檔案處理、API 呼叫
- **產出**: 寫一個簡單的 Python 腳本

### Level 2: LLM 應用 (Python 基礎完成後)
- [ ] OpenAI API 熟練使用
- [ ] Prompt Engineering 技巧
- [ ] 簡單的 Chat 應用
- **產出**: 做一個 CLI Chatbot

### Level 3: RAG 系統 (Level 2 完成後)
- [ ] Embedding 概念
- [ ] 向量資料庫 (Chroma)
- [ ] LangChain 基礎
- **產出**: 做一個可以問答自己文件的 RAG

### Level 4: AI Agent (Level 3 完成後)
- [ ] Function Calling
- [ ] Tool 整合
- [ ] Agent 架構
- **產出**: 做一個能執行任務的 Agent
