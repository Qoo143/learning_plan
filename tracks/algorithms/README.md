# 演算法

> 預計時間: 2-4 週
> 你的起點: 平均 1.2/5 分

---

## 🎯 為什麼要學？

1. **面試**: LeetCode 類型題目
2. **效能評估**: 知道程式會不會變慢
3. **問題解決**: 培養分解問題的能力

- [ ] 遞迴基礎
