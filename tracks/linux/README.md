## 🎯 為什麼要學？

1. **伺服器操作**: 大部分 server 都是 Linux
2. **Docker 基礎**: Docker 跑在 Linux 上
3. **部署維運**: ssh, 檔案管理, 服務管理

- [ ] 程序管理: ps, top, kill
